# داشبورد مدیریت Tetrashop

## معرفی
این داشبورد برای مدیریت و مانیتورینگ ۲۳ سرویس ابری Tetrashop طراحی شده است.

## ویژگی‌ها
✅ نمایش وضعیت real-time سرویس‌ها  
✅ کنترل سرویس‌ها (شروع، توقف، راه‌اندازی مجدد)  
✅ نمودار توزیع سرویس‌ها  
✅ دسته‌بندی ۶ گانه سرویس‌ها  
✅ رابط کاربری فارسی و ریسپانسیو  

## راه‌اندازی

### پیش‌نیازها
- Gateway API باید روی پورت 3000 در حال اجرا باشد

### مراحل
1. Gateway را اجرا کنید:
\`\`\`bash
cd ~/tetrashop-projects
node core/gateway/server.js
\`\`\`

2. داشبورد را اجرا کنید:
\`\`\`bash
./scripts/start-dashboard.sh
\`\`\`

3. مرورگر را باز کنید:
\`\`\`bash
http://localhost:3001
\`\`\`

## توسعه
برای اضافه کردن سرویس جدید:
1. سرویس را در Gateway ثبت کنید
2. endpoint مربوطه را اضافه کنید
3. UI را در داشبورد به‌روز کنید
