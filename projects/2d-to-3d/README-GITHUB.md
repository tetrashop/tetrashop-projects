# 🎯 پروژه تبدیل تصویر به مدل ۳D

## 🌐 دسترسی آنلاین
- **مخزن GitHub**: https://github.com/tetrashop/tetrashop-projects
- **پوشه پروژه**: https://github.com/tetrashop/tetrashop-projects/tree/main/2d-to-3d

## 🚀 اجرای سریع
1. فایل `index.html` را دانلود کنید
2. در مرورگر باز کنید
3. تصویر خود را آپلود کنید
4. مدل ۳D واقعی را دریافت کنید

## 📁 ساختار پروژه
```

2d-to-3d/
├──index.html              # رابط کاربری اصلی
├──real-algorithm.js       # الگوریتم تبدیل
├──run.sh                 # اسکریپت اجرا
├──README.md              # مستندات
├──test-image.jpg         # تصویر نمونه
└──config.json           # تنظیمات

```

## 🎨 ویژگی‌ها
- تبدیل واقعی تصویر به مدل ۳D (نه شکل پیش‌فرض)
- نمایش real-time با Three.js
- خروجی فایل OBJ استاندارد
- رابط کاربری فارسی و مدرن
- سیستم تست و لاگ کامل
