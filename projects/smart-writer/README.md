# پروژه smart-writer

## توضیحات
این پروژه بخشی از مجموعه تتراشاپ است.

## ویژگی‌ها
- رابط کاربری مدرن
- طراحی واکنش‌گرا
- کدنویسی تمیز

## راه‌اندازی
فایل index.html را در مرورگر باز کنید.

## توسعه
برای توسعه پروژه:
1. فایل‌های HTML در ریشه
2. استایل‌ها در پوشه css/
3. اسکریپت‌ها در پوشه js/
4. تصاویر و فایل‌های رسانه در assets/
